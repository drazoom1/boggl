개인정보처리방침

시행일: 2026년 6월 21일

BOGGL(이하 "서비스")은 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 및 Google API Services User Data Policy를 준수합니다. 본 방침은 서비스가 어떤 정보를 수집·이용하는지 안내합니다.

1. 수집하는 정보

서비스는 Google 계정으로 로그인할 때 다음 정보를 받습니다.

  • 계정 기본 정보: 이름, 이메일 주소, 프로필 사진 (Google 로그인 시 제공)
  • YouTube 구독 정보: youtube.readonly 권한으로 이용자의 구독 목록을 조회하여, ‘혼잡스’ 채널 구독 여부만 확인합니다.

2. 이용 목적

  • 로그인 및 본인 식별
  • ‘혼잡스’ 채널 구독 인증을 통한 서비스 이용 자격 확인 및 기능 제공

3. YouTube 데이터 이용 및 제한적 사용(Limited Use)

서비스가 Google API로부터 받은 정보의 이용은 Google API Services User Data Policy(제한적 사용 요건 포함)를 준수합니다.

  • 구독 정보는 구독 여부 확인 용도로만 사용하며, 영상·재생목록 등 다른 데이터는 수집하지 않습니다.
  • 해당 데이터를 제3자에게 판매·공유하지 않습니다.
  • 광고 목적으로 사용하지 않습니다.

4. 보관 및 파기

  • 로그인 세션 및 구독 확인 결과는 이용자 브라우저(localStorage)에만 저장되며, 별도 서버에 저장하지 않습니다.
  • 로그아웃하거나 로그인 토큰이 만료되면 해당 정보는 삭제됩니다.

5. 제3자 제공

서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다.

6. 이용자 권리

이용자는 언제든지 로그아웃으로 데이터 사용을 중단할 수 있으며, Google 계정 설정(myaccount.google.com/permissions)에서 서비스의 접근 권한을 직접 철회할 수 있습니다.
